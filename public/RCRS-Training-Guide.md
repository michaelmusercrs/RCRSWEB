## Introduction

### Purpose
This training guide is designed to help all River City Roofing Solutions team members effectively use the company portal system. The portal centralizes operations including order management, inventory tracking, delivery coordination, and content management.

### Goals
- **Efficiency**: Streamline daily operations across all departments
- **Communication**: Keep everyone informed with real-time updates
- **Accountability**: Track progress and performance transparently
- **Training**: Help new team members get up to speed quickly

### Access Information
- **Portal URL**: `www.rivercityroofingsolutions.com/portal`
- **Compatible Browsers**: Chrome, Safari, Firefox, Edge (latest versions)
- **Mobile**: Fully responsive - works on phones and tablets
- **No App Required**: Just use your web browser

## Getting Started

### First Login
1. Navigate to the portal URL in your browser
2. Select your login type:
   - **Drivers**: Enter your 4-digit PIN
   - **Staff**: Enter your company email
3. Complete the training popup (first time only)
4. You'll be directed to your role's default dashboard

### Training Popup
When you first log in, you'll see a training popup showing:
- Your role-specific required training modules
- Your current progress percentage
- Top performers on the leaderboard
- Your rank in the company

**Options:**
- **"Go to Training"**: Opens the full training portal
- **"Close"**: Dismisses the popup for this session
- **"Don't show again"**: Prevents popup from appearing (can re-enable in Settings)

### Browser Tips
- **Bookmark** the portal page for quick access
- **Add to Home Screen** on mobile for app-like experience
- **Enable Notifications** for delivery and inventory alerts

## Points & Gamification System

### How Points Work
The portal features a gamification system to encourage training completion and daily engagement.

### Earning Points

| Action | Points Earned |
|--------|---------------|
| Complete a lesson | +10 points |
| Complete a module | +50 points |
| Complete ALL required training | +200 bonus |
| Daily login | +5 points |
| 7-day streak | +25 bonus |

### Streaks
- Log in daily to build your streak
- A streak counter shows next to your name
- Every 7 consecutive days earns a bonus
- Missing a day resets your streak to 0

### Leaderboard
The leaderboard shows:
- **Top 3 Podium**: Gold, Silver, Bronze positions
- **Your Position**: See who's above and below you
- **Points**: Total points earned
- **Streak**: Current consecutive day streak
- **Modules Completed**: Training progress

### Tips to Climb the Leaderboard
1. Complete all lessons in each module
2. Log in every day to build streaks
3. Don't miss weekends - they count too!
4. Complete your role's required training first
5. Help others learn (peer recognition coming soon)

## Admin Portal

### Overview
The Admin Portal is for website content management. Only admins and authorized staff have access.

### Blog Management
**Path**: Portal ‚Üí Admin ‚Üí Blog Posts

**Creating a Post:**
1. Click "Add New Post"
2. Fill in the title (will auto-generate slug)
3. Write content using the rich text editor
4. Add featured image
5. Select category
6. Set publish date
7. Click "Publish" or "Save Draft"

**Best Practices:**
- Use descriptive titles with keywords
- Include images every 2-3 paragraphs
- Add internal links to services
- Target 800-1500 words for SEO
- Update old posts regularly

### Team Management
**Path**: Portal ‚Üí Admin ‚Üí Team Members

**Adding a Member:**
1. Click "Add Team Member"
2. Upload professional headshot
3. Fill in name and role
4. Write brief bio (2-3 sentences)
5. Add contact info if applicable
6. Set order for display priority
7. Save changes

**Photo Guidelines:**
- Square format preferred
- Minimum 400x400 pixels
- Professional attire
- Neutral background
- Good lighting

### Image Gallery
**Path**: Portal ‚Üí Admin ‚Üí Images

**Uploading Images:**
1. Click "Upload Images"
2. Select multiple files (drag & drop supported)
3. Wait for upload to complete
4. Add titles and alt text for SEO
5. Organize into categories

**File Requirements:**
- Formats: JPG, PNG, WebP
- Max size: 10MB per file
- Recommended: 1200px wide for web
- Always compress before upload

### Service Areas
**Path**: Portal ‚Üí Admin ‚Üí Service Areas

**Adding a City:**
1. Click "Add Service Area"
2. Enter city name
3. Add zip codes served
4. Write location-specific content
5. Upload local images if available
6. Set as active/inactive
7. Save changes

## Manager Dashboard

### Overview
The Manager Dashboard is your command center for daily operations.

### Creating Orders
**Path**: Portal ‚Üí Manager ‚Üí New Order

**Steps:**
1. Select customer from list or add new
2. Choose order type (Delivery, Install, Service)
3. Add line items (materials, labor)
4. Set delivery date and time window
5. Assign to driver (optional)
6. Add special instructions
7. Submit order

### Tracking Deliveries
**Path**: Portal ‚Üí Manager ‚Üí Deliveries

**Status Types:**
- **Pending**: Awaiting assignment
- **Assigned**: Driver assigned, not started
- **In Transit**: Driver on the way
- **Delivered**: Completed successfully
- **Issue**: Problem occurred (see notes)

**Actions:**
- Click delivery to see full details
- Reassign drivers as needed
- View real-time location (when available)
- Contact driver directly

### Reports
**Path**: Portal ‚Üí Manager ‚Üí Reports

**Available Reports:**
- Daily delivery summary
- Weekly performance metrics
- Monthly material usage
- Driver efficiency rankings
- Customer satisfaction scores

## Inventory System

### Overview
Track materials, perform counts, and manage restock requests.

### Weekly Counts
**Path**: Portal ‚Üí Inventory ‚Üí Count

**Performing a Count:**
1. Start new count session
2. Go through each category
3. Enter actual quantities on hand
4. Note any damaged items
5. Submit count when complete
6. Review discrepancies

**Best Practices:**
- Count at the same time each week
- Double-check unusual variances
- Report damaged materials immediately
- Keep storage areas organized

### Restock Requests
**Path**: Portal ‚Üí Inventory ‚Üí Restock

**Creating a Request:**
1. Click "New Restock Request"
2. Select items needed
3. Enter quantities
4. Set priority (Normal, Urgent)
5. Add notes for supplier
6. Submit request

### Low Stock Alerts
The system automatically alerts when:
- Items fall below minimum thresholds
- Fast-moving items trend low
- Seasonal items need restocking

## Driver Portal

### Overview
The Driver Portal is your mobile hub for managing deliveries.

### PIN Login
- Your 4-digit PIN is private
- Don't share with anyone
- Contact office if forgotten
- Changes require manager approval

### Viewing Deliveries
**Path**: Portal ‚Üí Driver

Your screen shows:
- Today's assigned deliveries
- Customer names and addresses
- Delivery windows
- Special instructions
- Material lists

### Updating Status
**For Each Delivery:**
1. Tap "Start Delivery" when departing
2. Navigate to location (tap address for maps)
3. Complete delivery
4. Take photo of materials at site
5. Get customer signature if required
6. Tap "Complete Delivery"
7. Add notes if needed

### Taking Photos
**Required Photos:**
- Materials placed at delivery location
- Any damage noted
- Hard-to-find locations
- Customer signature screen

**Tips:**
- Ensure good lighting
- Include address/house number if possible
- Capture full material stack
- Take before leaving the site

### Handling Issues
**If Problems Occur:**
1. Stay calm and professional
2. Update status to "Issue"
3. Call office immediately
4. Document everything
5. Take photos of any damage
6. Let office handle customer complaints

## Troubleshooting

### Can't Log In
1. Check your PIN/email is correct
2. Clear browser cache
3. Try a different browser
4. Contact office for PIN reset

### Page Won't Load
1. Check internet connection
2. Refresh the page (F5 or pull down)
3. Clear browser cache
4. Try incognito/private mode

### Missing Data
1. Wait 30 seconds and refresh
2. Check your internet connection
3. Log out and log back in
4. Contact admin if persists

### Photo Upload Failed
1. Check file is JPG, PNG, or WebP
2. Ensure file is under 10MB
3. Check internet connection
4. Try resizing the image

### Delivery Status Not Updating
1. Check internet connection
2. Refresh the page
3. Try logging out and back in
4. Contact office to manually update
