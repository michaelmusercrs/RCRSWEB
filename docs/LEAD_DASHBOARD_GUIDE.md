## üìà Metrics Explained

### Lead Score (0-100)

**How it's calculated:**
- Base score: 50 points
- Phone provided: +20 points
- Detailed message (50+ words): +15 points
- Inspector preference: +10 points
- Urgent keywords: +10 points
- Business email: +5 points

**What the scores mean:**
- **90-100:** Exceptional lead - call immediately
- **80-89:** Hot lead - very high conversion probability
- **70-79:** Warm lead - good conversion potential
- **60-69:** Decent lead - follow up promptly
- **50-59:** Average lead - standard follow-up
- **0-49:** Cold lead - may require nurturing

### Conversion Rates

**Overall Conversion Rate:**
`Projects Won √∑ Total Leads √ó 100`

**Industry Benchmarks:**
- Excellent: 30-40%
- Good: 20-30%
- Average: 15-20%
- Needs Improvement: <15%

**Stage-Specific Rates:**
- Contact ‚Üí Schedule: Target 80%
- Schedule ‚Üí Quote: Target 90%
- Quote ‚Üí Win: Target 35%

### Response Time Impact

**Statistics:**
- <5 minutes: 21x better conversion
- <1 hour: 7x better conversion
- <24 hours: 2x better conversion
- >24 hours: Significantly reduced chance

**Target:**
100% of leads contacted within 1 hour

### Revenue Metrics

**Total Revenue:**
Sum of all won projects in period

**Average Project Value:**
`Total Revenue √∑ Projects Won`

**Revenue Per Lead:**
`Total Revenue √∑ Total Leads`

**Projected Monthly Revenue:**
Current revenue + (quoted projects √ó win rate)

## üÜò Troubleshooting

### Dashboard Not Loading

**Check:**
1. Internet connection
2. Browser cache (clear if needed)
3. API endpoint status
4. Error console (F12)

**Fix:**
- Refresh page
- Try different browser
- Contact support

### Leads Not Appearing

**Check:**
1. Form submission working?
2. Google Script deployed?
3. API connection active?
4. Data file permissions?

- Test form submission
- Verify Google Script URL
- Check server logs

**Check:**

2. Filters applied

- Reset filters
- Select correct period
